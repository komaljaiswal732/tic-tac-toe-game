# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**. The game allows two players to play alternately as **X** and **O**, automatically detects the winner, and provides options to reset or start a new game.

## 🚀 Features

- Two-player gameplay (X and O)
- Automatic winner detection
- Reset Game functionality
- New Game option
- Responsive and clean user interface
- Winning message display
- Prevents overwriting already selected boxes

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 Game Rules

- Player **O** starts the game.
- Players take turns placing their marks.
- The first player to align three identical marks horizontally, vertically, or diagonally wins.
- If all boxes are filled and no player wins, the game ends in a draw.
